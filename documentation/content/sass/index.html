Files with a leading underscore in the name are not placed into the public folder, but can still be used as @import dependencies. For more information, see the "Partials" section of Sass Basics.

Files with the scss extension use "Sassy CSS" syntax, while files with the sass extension use the "indented" syntax. Zola will return an error if a scss and sass file exist with the same base name in the same folder to avoid confusion -- see the example above.
